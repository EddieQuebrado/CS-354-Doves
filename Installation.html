Python Installation

The default installation of python comes with a documentation file, an IDE (IDLE), a GUI toolkit (tkinter), a package manager (pip), a test suite, and a command-line Python interpreter. Many mainstream operating systems and distributions already have Python installed, but if yours doesn't, it's a snap to install. Python has pre-built installation executables for Windows, MacOS, and Linux. If you prefer, you can also download the source code and build it yourself. There are two mainstream versions of python (2 & 3), but for new installations, version 3 is strongly recommended.

Windows

Installing python for Windows is as simple as installing any other application.

  • Navigate to Python's releases for Windows.
  • Download the latest version.
  • Execute the installer.
  • (optional) if you want to use Python from the command line more easily, choose the option to add Python to your PATH variable.
  • Choose install now, and then walk through the installation wizard.

If you prefer to use a package manager, Python is also available on Chocolatey, via:

choco install python

MacOS

Similarly, installing Python on MacOS is a breeze.

  • Navigate to Python's releases for MacOS.
  • Download the latest version.
  • Execute the installer.
  • Choose install now, and then walk through the installation wizard.

If you prefer to use a package manager, Python is also available on Homebrew, via:

brew install python

Linux

Installing Python on Linux is just as easy, but depends on your distribution. Chances are, Python is already installed and you can just start using it!

The simplest way is to use your distribution's built in package manager. Here are a few examples:

Ubuntu (Apt)

$ sudo apt-get update

$ sudo apt-get install python3.6

Fedora (Yum)

$ sudo yum install python3

Arch (Pacman)

$ pacman -S python

Development tools and utilities are available in the same manner.

Building from source

If you need (or want) the most bleeding edge version of Python, you'll most likely need to build it from the source code. These instructions are for Linux but can be adapted to other operating systems as well.

  • First, clone the source code available here.

    • $ git clone https://github.com/python/cpython

  • Then, run the configuration script:

    • $ ./configure

    Note that the configuration script has many options to customize the installation. Learn more by running:

    $ ./configure --help

  • We can then make and install Python using:

    • $ make

    $ make test

    $ sudo make install
