Introductory Python Programs

This program will roll a random 6 sided dice and print the result:

import random #imports the module which allows for the generation of random numbers

print("Rolling a dice.") #simple text output to notify the user what we're doing
print(random.randint(1,6)) #printing a randomly generated integer between 1 and 6, to represent the 6 sides of a regular dice
